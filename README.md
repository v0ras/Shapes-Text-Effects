# 🎨 Shapes & Text Effects

Interactive creative tool for generating and manipulating visual elements – shapes and text. Built with modern JavaScript (ES6+), HTML5, and CSS3. No jQuery or legacy dependencies.

## ✨ Features

- ✅ Create **shapes** (width, height, background, border, border-radius)
- ✅ Create **text elements** (custom font size, color, content)
- ✅ **Drag & drop** any element anywhere on canvas
- ✅ **Rotate** elements: double-click + mouse wheel
- ✅ **Delete** elements by dragging them to the trash bin
- ✅ Real-time **element counter**
- ✅ Change **background color** of working area
- ✅ **Help window** with toggle checkbox
- ✅ Fully responsive, clean modern UI

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, Grid, transitions)
- JavaScript (ES6+, no frameworks)

## 🚀 How to use

1. Open `index.html` in any modern browser.
2. Adjust shape/text parameters.
3. Click **"Sukurti elementą"** to add.
4. Double-click + scroll to rotate.
5. Drag element to trash bin to delete.

## 📁 Files

- `index.html` – structure
- `style.css` – styling
- `script.js` – all interactivity (vanilla JS)
